Generates a random sparse matrix of a specified size.

This function generates a sparse matrix of size rows × cols, where each element is assigned with a probability of 1 - random_factor. Non-zero elements are assigned random values uniformly distributed between min_val and max_val.

Template Parameters
TThe type of matrix elements (e.g., int, double, std::complex).
Parameters
rowsThe number of rows in the matrix.
colsThe number of columns in the matrix.
random_factorThe probability that an element will be zero (sparsity factor).
min_valThe minimum value for non-zero elements.
max_valThe maximum value for non-zero elements.
Returns
A 2D vector representing the sparse matrix.

Generates a random vector of a specified size.

This function generates a vector of size n, where each element is a random number uniformly distributed between min_val and max_val. The type of the vector elements can be integral (e.g., int), floating-point (e.g., double), or complex (std::complex<double>).

Template Parameters
TThe type of vector elements (e.g., int, double, std::complex).
Parameters
nThe size of the vector.
min_valThe minimum value for elements.
max_valThe maximum value for elements.
Returns
A vector of randomly generated values.
