Detailed Description

template<typename T, StorageOrder Order>
class algebra::Matrix< T, Order >

A sparse matrix class with optional compression.

Template Parameters
TType of the matrix elements.
OrderStorage order (RowMajor or ColumnMajor).

Provides efficient storage and operations for sparse matrices, supporting both COO and compressed (CSR/CSC) formats. Includes functionalities like update, compression, decompression, resizing, and matrix-vector multiplication (serial and parallel).

template<typename T, StorageOrder Order>
std::vector< T > algebra::Matrix< T, Order >::product_by_vector ( const std::vector< T > & v) const

Multiplies the (possibly uncompressed) matrix by a vector.

Multiplies the matrix by a vector v. If the matrix is compressed, it uses either parallel or regular multiplication based on the number of rows (CSR case) or columns (CSC case). For uncompressed matrices in COO format, it performs the multiplication by iterating over sparse data. Inputs: v - a vector of type T, Outputs: a vector of type T with the result of the multiplication.
