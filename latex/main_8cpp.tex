int main (\begin{DoxyParamCaption}{}{}\end{DoxyParamCaption})}



Main function to execute various tests. 

This function presents a menu to the user to select a specific test case to run. Based on the user\textquotesingle{}s input, it calls the corresponding test function. The available test cases include matrix-\/vector multiplication, speed tests for compressed and uncompressed matrices, matrix norm calculation, and more.

The tests include\+:
\begin{DoxyEnumerate}
\item Matrix-\/\+Vector Multiplication Test 1
\item Compressed Parallel vs Unparallel Speedtest
\item Compressed vs Uncompressed Speedtest
\item \doxylink{classalgebra_1_1_matrix}{Matrix} Market Speedtest (with a sample matrix file)
\item Complex \doxylink{classalgebra_1_1_matrix}{Matrix} Times Vector Test
\item \doxylink{classalgebra_1_1_matrix}{Matrix} Norm Test (int, Row\+Major)
\item \doxylink{classalgebra_1_1_matrix}{Matrix} Resize Test
\item Diagonal View Test
\item Full Multiplication Speedtest
\end{DoxyEnumerate}

The user is prompted to select a test case by entering a number between 1 and 9.

\begin{DoxyReturn}{Returns}
0 on successful execution of the selected test, or 1 if an invalid choice is entered. 
\end{DoxyReturn}
