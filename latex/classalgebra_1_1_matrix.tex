\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename T, \mbox{\hyperlink{namespacealgebra_af7f291bc9289c8ac99068a8ae4873108}{Storage\+Order}} Order$>$\newline
class algebra\+::\+Matrix$<$ T, Order $>$}
A sparse matrix class with optional compression. 


\begin{DoxyTemplParams}{Template Parameters}
{\em T} & Type of the matrix elements. \\
\hline
{\em Order} & Storage order (Row\+Major or Column\+Major).\\
\hline
\end{DoxyTemplParams}
Provides efficient storage and operations for sparse matrices, supporting both COO and compressed (CSR/\+CSC) formats. Includes functionalities like update, compression, decompression, resizing, and matrix-\/vector multiplication (serial and parallel).
